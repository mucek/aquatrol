#### \Casing\: Casing files
#### \Docs\: Manuals and other documentation
#### \Firmware\: Firmware for the board (Bascom), main program (Aquatrol.hex) and bootloader (BOOT_Aquatrol.hex)
#### \IMGs\: Images
#### \PCB\: PCB project (Altium Designer 20)
